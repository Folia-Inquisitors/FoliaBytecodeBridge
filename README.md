# FoliaBytecodeBridge

Experimental plugin that reroutes unsafe Bukkit/Paper plugin calls to Folia-safe scheduler, entity, and region ownership paths.

## Building instructions

```bash
mvn clean install
```

## Official Discord

https://discord.gg/aT9z7q7hX8

## Documentation

See the `docs/` folder for architecture notes, route families, diagnostics, probes, and smoke-test details.

The synthetic compatibility/event-path model is documented at [`docs/SYNTHETIC_COMPATIBILITY_MODEL.md`](docs/SYNTHETIC_COMPATIBILITY_MODEL.md).

The previous detailed README was preserved at [`docs/TECHNICAL_README.md`](docs/TECHNICAL_README.md).


## Disclaimer

This project is experimental. It can test plugin compatibility on Folia, but it does not prove that any plugin is region-thread safe.

AI  was used during development, testing, and documentation.
